# cussli

> cussli is a command-line interface to cuss

## requirements, node

## installation

```bash
npm -g install cussli
```

*or...*


```bash
yarn global add cussli
```

## usage

Pass a word to cussli as the first argument, and the likelihood of the word
being a cuss-word is returned.

```bash
cussli [word]
```

## expected output

- unknown
- unlikely
- likely
- most-likely
